### Installation

```bash
# 1. Clone the repository.
git clone {REPO_URL}

# 2. Enter your newly-cloned folder.
cd {PROJECT_DIR}

# 3. Install dependencies. Make sure node is installed
npm i 

# 4. Start the development server.
npm start

# To start the production server.
npm start:production
```
